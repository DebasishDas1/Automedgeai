**How to use this:**

Paste into Cursor with your existing codebase open. Since you already have the homepage built in the Parker style, Cursor will match the component patterns exactly. The prompt references your existing CSS variables (`--accent`, `--background`, `font-display`) so everything inherits automatically.

For the other 3 verticals, change only these variables:
